一、在firefox中安装flashgot扩展

flashgot是firefox的一个扩展，在联网的情况下，可以在firefox中的附加组件中搜索flashgot，然后安装。

二、在flashgot中配置不同的下载管理器

1、配置multiget为下载器

multiget为国人开发的多线程下载工具，可惜现在作者已经停止维护了。flashgot中的配置如下：

可执行路径：

$HOME/opt/bin/multiget

参数模板：

[url=URL]空格[refer=REFERER]

2、配置lftp为下载器

lftp为Linux下一款非常好用的开源ftp工具，支持ftp,http,sftp等多种协议，支持多线程端点续传。flashgot中的配置如下：

可执行路径：

/usr/bin/xterm

参数模板：

-e /usr/bin/lftp -c pget -n -c 8 [URL] -o [FOLDER]

xterm比较丑陋，若用gnome-terminal，则参数模板可设置为：

--hide-menubar -x /usr/bin/lftp -c pget -n 8 -c [URL] -o [FOLDER]

注意：pget命令后的-c是开启续传的功能

3、配置uget+aria2c为下载器

aria2 是 Linux 下一个不错的高速下载工具，支持从多个地址或者从一个地址的多个连接来下载同一个文件，而且还支持 BitTorrent 协议。但他是个命令行工具，搭配上uget就完美了。需要在uget中启用aria2c插件。Arguments参数可能根据不同的aria2c版本需要修改下，1.13.0版支持的参数设置如下图：
