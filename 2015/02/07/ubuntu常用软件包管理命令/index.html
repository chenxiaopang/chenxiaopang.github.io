1、查看软件包安装内容

1
2
3
4

 # dpkg -L [packagename] 

 //查看安装binutils包会安装哪些文件
 # dpkg -L binutils

2、列出指定包的安装状态信息

# dpkg -l [packagename]

3、查找文件所属包

1
2
3

 # dpkg -S [filename]
 # dpkg -S /usr/bin/objdump
 binutils: /usr/bin/objdump

4、查找软件

# apt-cache search [正则表达式]

5、查询软件包依赖哪些包

# apt-cache depends [packagename]

6、查询软件包被哪些包依赖

# apt-cache rdepends [packagename]

7、查询依赖和被依赖的包

显示依赖的包（forward dependency）和被依赖的包（reverse dependency）

# apt-cache showpkg [packagename]

8、增加一个光盘源

# apt-cdrom add

9、更新软件源索引

# apt-get update

10、安装软件包

1
2

 # apt-get install [packagename]
 # apt-get install [packagename] --print-uris     //仅仅打印出URL，而不实际安装

11、忽略依赖关系强制安装

# dpkg -i --force-depends packagename.deb

或

# dpkg -i --ignore-depends=packageneme

12、安装已经下载的包，忽略缺少的包

# apt-get install [packagename] --ignore-missing --no-download

13、将系统中所有软件升级到最新版本

# apt-get upgrade

14、将操作系统连同所有软件升级到最新版本

# apt-get dist-upgrade

15、下载指定软件包的源文件

# apt-get source [packagename]

16、备份当前系统安装的所有包的列表

1

# dpkg --get-selections | grep -v  deinstall > ~/install_list

17、从备份的安装包列表文件恢复选择并安装所有包

1
2

 # dpkg --set-selections < install_list
 # apt-get dselect-upgrade

18、卸载软件包

# apt-get remove [packagename] # apt-get purge [packagename] //卸载软件并删除配置文件

19、清理旧版本的软件缓存

# apt-get autoclean

20、清理所有软件包缓存

# apt-get clean

21、删除系统不再使用的孤立软件包

# apt-get autoremove

22、安装软件包到指定目录

# dpkg -i --instdir=[install_dir] *.deb
