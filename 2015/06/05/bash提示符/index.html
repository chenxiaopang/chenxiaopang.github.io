Bash提示符

2015-06-05
Linux Bash
约 485 字 预计阅读 1 分钟

Bash有四种提示符

一、基本提示符PS1

即$符号，是默认的基本提示符，当Shell运行在交互模式下时，该提示符会出现在屏幕上，可以设置为其它符号。

显示PS1设置

1
2

[cb@cb:16:36:23:~]$ echo $PS1
[\u@\h:\t:\W]$

设置PS1 $ PS1="[\u@\h:\t:\W]\$"

用于设置PS1的常用的特殊转义序列：

转义符 意义
\t 以HH:MM:SS格式显示时间
\d 以"星期/月/日"格式显示时间
\w 当前工作目录的全名
\W 当前工作目录的基本名
\u 当前用户的用户名
\h 主机名
$ 如果有效UID=0，则为#，否则为$

二、次提示符PS2

当没有完成输入或期望更多的输入时就会出现这个提示符，默认为>符号

显示PS2

1
2

# ehco $PS2
>

重新设置PS2

#PS2="-->"

                wed) echo "Wednesday";;
                exit) exit;;
        esac
done

243

$ export PS4='$0.$LINENO+ '  //$0显示当前的脚本名, $LINENO显示的当前的行号

$ ./ps4.sh
./ps4.sh.3+ echo 'PS4 demo script'
PS4 demo script
./ps4.sh.4+ ls -l /etc/
./ps4.sh.4+ wc -l
