一、安装mysql服务器和客户端

$ sudo apt-get install mysql-server mysql-client

二、启动mysql服务

$ sudo /etc/init.d/mysql start

三、连接mysql服务器

1、root超级用户登录

# mysql -u root -p

2、普通用户登录

$ mysql -u bookorama -p

3、访问数据库表

1
2
3
4

mysql>show databases;
mysql>use books;
mysql>show tables;
mysql>select * from books;

四、停止mysql服务

$ sudo /etc/init.d/mysql stop

五、禁止匿名登陆

1
2
3
4
5

# mysql -u root -p
mysql>use mysql;
mysql>delete from user where User='';
mysql>quit;
# mysqladmin -u root -p reload

六、更改root用户密码

$ mysqladmin -u root -p password 'newpassword'

七、查看数据库存放位置

# mysqladmin -u root -p variables|grep datadir

八、删除数据库

1
2

mysql>drop table cmd;
mysql>drop tables cmd,cmd_all,proptags;

注意：删除表时，如果没有外键约束，才能删除成功

十一、函数

十二、创建用户并授权

创建admin用户，该用户拥有所有数据库，包括mysql数据库的所有权限，并且能对其它用户授权。

1
2
3
4

# mysql -u root -p
mysql>grant all on *.* to admin with grant option;
mysql>show grants;
mysql>show grants for admin;

十三、给数据表授权

mysql>grant update (password) on cmddb_comm.system_users to test;

十四、修改数据库列

change需要加上旧列和新列的名称，即使不改变名称，也要写上

mysql>alter table cmds change cmd_syntax1 cmd_syntax varchar(150) not null;

modify不能改变列的名称

mysql>alter table cmds modify param_description varchar(600) DEFAULT null;

十五、修改数据表名

mysql>alter table dev_sub_series rename to device_sub_series;
