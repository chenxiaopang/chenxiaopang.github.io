Linux下串口通信工具minicom的用法

2016-09-16
Linux Tools
约 148 字 预计阅读 1 分钟

一、查看串口设备

将USB转串口线插入交换机Console口后，执行命令

$ ll /dev/ttyUSB*

二、连接串口设备

$ sudo minicom -D /dev/ttyUSB0

三、设置串口参数

在minicom中按CTRL+A，O，配置串口速率9600,8位，无奇偶校验，保存
