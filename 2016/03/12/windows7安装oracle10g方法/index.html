Windows 7安装Oracle 10g的方法

Windows 7下安装Oracle 10g提示“程序异常终止，发生未知错误”的解决方法

一.修改Oracle配置文件refhost.xml

修改\Oracle 10G\database\stage\prereq\db\refhost.xml

在</SYSTEM> <CERTIFIED_SYSTEMS>后面添加

1
2
3
4

<!--Microsoft Windows 7-->
<OPERATING_SYSTEM>
<VERSION VALUE="6.1"/>
</OPERATING_SYSTEM>

二.修改Oracle配置文件oraparam.ini

到install目录中找到oraparam.ini文件，把

#Windows=4.0,5.0,5.1,5.2

修改成

#Windows=4.0,5.0,5.1,5.2,6.1

并在后面添加

1
2
3
4
5
6

[Windows-6.1-required]
#Minimum display colours for OUI to run
MIN_DISPLAY_COLORS=256
#Minimum CPU speed required for OUI
#CPU=300
[Windows-6.1-optional]

三.以管理员身份安装

右键setup.exe，选择属性->兼容性->以兼容模式运行这个程序 windows xp（ service pack 3），以管理员身份运行安装就可以了。
