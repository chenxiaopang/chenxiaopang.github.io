Linux上的防病毒软件ClamAV

2016-04-04
Linux Tools
约 436 字 预计阅读 1 分钟

文章目录

Clam AntiVirus（ClamAV）是免费而且开放源代码的防毒软件，软件与病毒码的更新皆由社群免费发布。目前ClamAV主要是使用在由Linux、FreeBSD等Unix-like系统架设的邮件服务器上，提供电子邮件的病毒扫描服务。ClamAV本身是在文字接口下运作，但也有许多图形接口的前端工具可用，另外由于其开放源代码的特性，在Windows与Mac OS X平台都有其移植版。

1、安装clamav

1
2
3
4
5

$ sudo apt-get install clamav clamav-docs
$ sudo apt-get install clamav-daemon                             //安装clamav服务
$ sudo apt-get install clamtk                                                 //安装图形前端
$ sudo apt-get install clamtk-nautilus 				                //安装nautilus右键支持查毒
$ sudo apt-get install libclamunrar6                                   //支持rar文件查毒

2、升级病毒库

1

$ sudo freshclam

3、查看版本

1

$ clamscan -V

4、查毒

1

$ clamav -r /home/cb/software     //-r递归扫描目录

5、注意

  • clamav只能查毒，不能杀毒，但可以删除或隔离被病毒感染的文件
  • 病毒库文件位于/var/lib/clamav目录下，可备份并拷贝到其它机器上用于离线升级
  • 可结合at或cron服务定时扫描系统

以上在Ubuntu 12.04上测试通过。
