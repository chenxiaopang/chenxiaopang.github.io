一、安装

1、从MELPA Stable仓库中安装

(1) 配置.emacs

1
2
3
4

(require 'package)
(add-to-list 'package-archives
             '("melpa-stable" . "https://stable.melpa.org/packages/"))
(package-initialize)

(2) 安装markdown-mode包

M-x package-install RET markdown-mode RET

或者

M-x package-list-packages RET

找到markdown-mode包RET，点击Install即可。

(3) 全局安装

步骤2的安装会安装到~/.emacs.d/下，按下面的步骤拷贝到全局目录中，所有用户可用（在ubuntu 18.04下markdown-mode 2.4通过）。

1
2
3
4
5
6
7
8

$ cd /usr/share/emacs/site-lisp/elpa-src
$ sudo rm -rf *
$ sudo cp -r ~/.emacs.d/elpa/markdown-mode-2.4/ .

$ cd /usr/share/emacs/25.2/site-lisp/elpa
$ sudo ln -fs /usr/share/emacs/site-lisp/elpa-src/markdown-mode-2.4/markdown-mode-pkg.el markdown-mode-pkg.el
$ sudo ln -fs /usr/share/emacs/site-lisp/elpa-src/markdown-mode-2.4/markdown-mode.el markdown-mode.el
$ sudo ln -fs /usr/share/emacs/site-lisp/elpa-src/markdown-mode-2.4/markdown-mode-autoloads.el markdown-mode-autoloads.el

OK!

(setq markdown-command "pandoc") ;; 使用 pandoc 进行 markdown 导出，该命令支持更多 markdown 特性
(setq markdown-split-window-direction 'right) ;; 动态预览窗口在右侧打开
;; 设置预览使用的浏览器

; 在原有默认配置中，使用 eww 浏览器
; (setq markdown-live-preview-window-function 'markdown-live-preview-window-eww)

; 可将其调整，使用 eaf 浏览器，但需要先安装EAF-Emacs Application Framework
; @limitation：1）预览的 Buffer 名称是变化的；2）页面会上下滚动
(defun markdown-live-preview-window-eafbrowser (file)
  "Preview FILE with eaf.
To be used with `markdown-live-preview-window-function'."
  (if (get-buffer "*eaf*")
      (kill-buffer "*eaf*"))
  (eaf-open-browser file)
  (get-buffer "*eaf*"))
(setq markdown-live-preview-window-function 'markdown-live-preview-window-eafbrowser)
